Command Line Toolsのインストール

Command Line Toolsとは、Mac OS上でソフトウェアをコンパイルするために必要なライブラリやコマンドが収録されたライブラリです。

一般的に、Mac OS上でソフトウェア開発を行う場合は必要になるツールです。

手順

[ターミナル] （[アプリケーション]の[ユーティリティ]）を立ち上げます。

 

ターミナルに、以下のように入力して[Enter]キーを押します。

% xcode-select --install

 

そうすると、以下のようなウィンドウが立ち上がるので、[インストール]ボタンを押し、ダイアログ通りに進めます。

 

およそ30分程度でインストールが完了します。
